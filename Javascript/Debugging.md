<h1>Debugging</h1>VSCODE

- Debug 탭에서 톱니바퀴를 누르면 launch.json 파일 생성

```
"configurations": [
        {
            ...
            "program": "${workspaceFolder}\\${filename}"
        }
    ]
```

- 설정이 끝나고 F5를 누르면 해달 file에 대한 디버깅이 시작된다.



이하 추후 추가

- breakpoints란
- watch사용법
- call stack 의 의미
- Step over / Step into/ Step out
